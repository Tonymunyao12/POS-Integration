COMPONENTS OF POS SERVER ARCHITECTURE:

1. Client Layer - these are the devices that interact with the system. - i.e. Android POS devices, web dashboards, mobile banking apps.
2. API Gateway - acts as the front end of the backend.
    flow: POS Device -> API Gateway -> Backend Services
3. Microservices Layer - its best practice to break down logic into microservices.
        Authentication  -  Login, tokens
        Payment Service  -  Transaction processing
        Device Service  -  POS device management
        Merchant Service  -  Merchant management
        Settlement Service  -  Bank Settlement
        Reconciliation Service  -  Transaction matching

4. Payment Processing  -  it receives the transaction
